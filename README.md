## Problem Statement

Managing student and faculty records manually can be time-consuming and error-prone. The objective of this project is to develop a console-based Education Management System using Java that allows administrators to manage records, faculty to update student marks, and students to view their academic information.

## Technologies Used

* Java JDK 8 or above
* Object Oriented Programming
* Console Based Application

## OOP Concepts Used

* Inheritance
* Classes and Objects
* Encapsulation
* Method Overriding
* Arrays of Objects

## Algorithm

1. Start the application.
2. Enter total student and faculty strength.
3. Admin logs in using the predefined password.
4. Admin adds student and faculty details.
5. Students log in using Name and USN to view their details.
6. Faculty log in using Name and ID.
7. Faculty can view their profile and update student CGPA using USN.
8. Updated information is stored during program execution.
9. User can exit the application through the main menu.

## How To Run

### Step 1 : Compile

```bash
javac EducationSystem.java
```

### Step 2 : Run

```bash
java EducationSystem
```
